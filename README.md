# AAP Søknad

Søknad for arbeidsavklaringspenger med idporten innlogging (wonderwall). Skrevet i NextJS.

Kom i gang lokalt:

```
yarn
yarn run dev
```

Push til main bygges og deployes til dev-gcp.
